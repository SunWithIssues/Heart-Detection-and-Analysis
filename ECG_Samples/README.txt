Directory INFO
--------------

The Baseline directory holds CSV files that monitor the baseline frequency of the software/hardware, that is without any item (e.g. human, trembling, etc) attached.

The New_Data directory holds sub-directories that contain CSV files of the monitoring of each test subject. The names of the test subjects have been hidden in the format "test_sub#", where "#" stands for a number. In total, we had 5 test subjects. "test_sub2" was monitor twice, because her heartbeat is hard to find. The second trial for her is called "test_sub2.1".
	Female test subjects: 2
	Male test subjects: 1, 3, 4, 5
